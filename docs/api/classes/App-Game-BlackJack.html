BlackJack.php : 15

Black Jack game where the player plays 1 to 3 hands against the bank, betting money from a personal balance.

Black Jack rules: the bank draws to 17, a natural blackjack pays 3:2, an equal value is a push (stake returned), otherwise 1:1.
